# 部署指南

## 本地运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 访问 `http://localhost:8000`

## 云服务配置

### 步骤1：创建项目
- 访问相关云服务平台
- 创建新项目
- 启用认证和数据库服务

### 步骤2：配置认证
- 启用邮箱/密码认证
- 启用第三方登录选项

### 步骤3：数据库设置
- 创建实时数据库
- 配置适当的访问规则
- 设置数据库索引

### 步骤4：更新配置
- 将项目配置信息复制到 `firebase-config.js` 文件中

### 步骤5：部署网站
- 使用平台提供的部署工具
- 或通过命令行工具进行部署

## 安全注意事项
- 确保设置正确的数据库访问规则
- 定期更新依赖库
- 监控异常访问情况

## 常见问题
- 如遇连接问题，请检查网络设置
- 如遇认证失败，请确认账号密码正确
- 如遇同步问题，请检查网络连接状态
