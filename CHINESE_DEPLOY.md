## 快速开始

### 1. 本地运行

直接在浏览器中打开 `index.html` 文件即可使用。

### 2. 本地服务器运行

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`

## 常见问题

### Q: 数据会丢失吗？
A: 数据存储在浏览器本地，清除浏览器数据会导致数据丢失。建议定期导出备份。

### Q: 支持多设备同步吗？
A: 当前版本为本地版，多设备同步需要部署服务器端。

### Q: 微信登录是真实的吗？
A: 当前为模拟实现，实际部署需要集成微信开放平台。

### Q: 可以在手机上使用吗？
A: 完全支持，已针对移动端优化。

### Q: 需要后端服务器吗？
A: 不需要，当前版本为纯前端实现。
